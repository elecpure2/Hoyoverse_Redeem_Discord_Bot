## 1. Python 3.11 설치

### 1-1. Python 다운로드
1. **Python 공식 웹사이트 접속**
   - 브라우저에서 https://www.python.org/downloads/ 접속
   - 또는 직접 링크: https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe

2. **Python 3.11.9 다운로드**
   - "Download Python 3.11.x" 버튼 클릭
   - 파일 저장 위치 기억하기

### 1-2. Python 설치
1. **다운로드한 설치 파일 실행**
   - `python-3.11.9-amd64.exe` 더블클릭

2. **⚠️ 중요: 설치 옵션 설정**
   - ✅ **"Add Python 3.11 to PATH"** 체크박스 **반드시 선택**
   - "Install Now" 클릭

3. **설치 완료 대기**
   - "Setup was successful" 메시지 확인
   - "Close" 클릭

### 1-3. 설치 확인
PowerShell을 열고 다음 명령어 실행:
```powershell
python --version
```
**예상 출력:** `Python 3.11.9` (또는 3.11.x)

만약 명령어가 작동하지 않으면:
- PowerShell을 **완전히 종료**하고 다시 열기
- 컴퓨터 재부팅 후 다시 시도

## 2. 프로젝트 의존성 설치

### 2-1. PowerShell에서 프로젝트 폴더로 이동
```powershell
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"
```

### 2-2. pip 업그레이드 (권장)
```powershell
python -m pip install --upgrade pip
```

### 2-3. 필수 패키지 설치
```powershell
pip install -r requirements.txt
```

**설치되는 패키지:**
- discord.py (Discord 봇 라이브러리)
- aiohttp (비동기 HTTP 클라이언트)
- beautifulsoup4 (웹 스크래핑)
- lxml (XML/HTML 파서)
- google-generativeai (Gemini AI)
- python-dotenv (환경 변수 관리)

### 2-4. 선택적 패키지 설치 (Hakushin)
```powershell
pip install git+https://github.com/thesadru/hakushin.git
```
> **참고:** 이 패키지는 선택사항입니다. 설치하지 않아도 봇의 대부분 기능은 작동합니다.

### 2-5. 설치 확인
```powershell
pip list
```
위 명령어로 설치된 패키지 목록을 확인할 수 있습니다.

## 3. 환경 변수 설정

봇을 실행하려면 **Discord 봇 토큰**과 **Gemini API 키**가 필요합니다.

### 3-1. .env 파일 확인
프로젝트 폴더에 `.env` 파일이 있는지 확인:
```powershell
Get-Item .env
```

### 3-2. .env 파일 편집
메모장이나 텍스트 에디터로 `.env` 파일을 열고 다음 형식으로 작성:

```env
DISCORD_TOKEN="여기에_실제_디스코드_봇_토큰_입력"
GEMINI_API_KEY="여기에_실제_Gemini_API_키_입력"
```

> **⚠️ 중요:** 
> - 따옴표 안에 실제 토큰과 API 키를 입력하세요
> - 이전 컴퓨터의 `.env` 파일을 복사해도 됩니다
> - 토큰/키를 잃어버렸다면:
>   - Discord 토큰: https://discord.com/developers/applications
>   - Gemini API: https://makersuite.google.com/app/apikey

### 3-3. .env 파일이 없는 경우
`.env.example` 파일을 복사하여 생성:
```powershell
Copy-Item .env.example .env
notepad .env
```
그 다음 위의 3-2 단계를 따라 편집하세요.

## 4. 봇 테스트 실행

백그라운드 서비스로 등록하기 전에 봇이 정상 작동하는지 테스트합니다.

### 4-1. 수동 실행
```powershell
python main.py
```

### 4-2. 정상 작동 확인
다음과 같은 메시지가 출력되면 성공:
```
🔄 Cog 로딩 중...
  ✅ cogs.fortune 로드 완료
  ✅ cogs.gacha 로드 완료
  ...
🚀 봇 시작 중...
✅ BotName#1234 로그인 완료!
📡 X개 서버에서 활동 중
✅ XX개 슬래시 명령어 동기화 완료
```

### 4-3. 봇 종료
테스트가 끝나면 `Ctrl + C`를 눌러 봇을 종료합니다.

### 4-4. 문제 발생 시
- **토큰 오류**: `.env` 파일의 토큰이 올바른지 확인
- **모듈 오류**: 2단계로 돌아가 패키지 재설치
- **기타 오류**: 하단 "문제 해결" 섹션 참고

## 5. 백그라운드 서비스 설정

봇이 정상 작동하는 것을 확인했다면, 이제 백그라운드에서 자동으로 실행되도록 설정합니다.

### 5-1. PowerShell 관리자 권한으로 실행
1. 시작 메뉴에서 "PowerShell" 검색
2. **마우스 우클릭** → "관리자 권한으로 실행"
3. 프로젝트 폴더로 이동:
   ```powershell
   cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"
   ```

### 5-2. 서비스 설치 스크립트 실행
```powershell
.\scripts\install_service.ps1
```

**예상 출력:**
```
✅ 작업 스케줄러에 'HoyoBot_AutoStart' 등록 완료!
✅ 컴퓨터 부팅 시 자동으로 봇이 시작됩니다.
```

### 5-3. 봇 시작
```powershell
.\scripts\control_bot.ps1
```
메뉴에서 **"1. 봇 시작"** 선택

### 5-4. 자동 시작 확인
- 컴퓨터를 재부팅하면 봇이 자동으로 시작됩니다
- 작업 관리자 → 세부 정보 탭에서 `python.exe` 프로세스 확인 가능

## 6. 봇 제어 방법

### 6-1. 제어 패널 사용 (추천)
```powershell
.\scripts\control_bot.ps1
```

**제공되는 기능:**
1. 봇 시작
2. 봇 중지
3. 봇 재시작
4. 봇 상태 확인
5. 로그 파일 보기
6. 로그 실시간 모니터링
7. 작업 스케줄러 상태 확인
8. 종료

### 6-2. 로그 확인
- **일반 로그**: `bot.log` 파일
- **에러 로그**: `bot_error.log` 파일 (백그라운드 실행 시)

실시간 로그 보기:
```powershell
Get-Content bot.log -Wait -Tail 50
```

### 6-3. 서비스 제거 (필요시)
자동 시작이 더 이상 필요 없으면:
```powershell
.\scripts\uninstall_service.ps1
```

## 7. 문제 해결

### 문제 1: Python 명령어가 작동하지 않음
**증상:** `python : 'python'은(는) 내부 또는 외부 명령...`

**해결방법:**
1. Python이 PATH에 추가되었는지 확인:
   ```powershell
   $env:PATH -split ';' | Select-String python
   ```
2. 없다면 Python을 재설치하고 "Add to PATH" 옵션 선택
3. PowerShell을 완전히 종료하고 다시 열기

### 문제 2: pip 설치 중 오류
**증상:** `error: Microsoft Visual C++ 14.0 is required`

**해결방법:**
1. Microsoft C++ Build Tools 설치:
   - https://visualstudio.microsoft.com/visual-cpp-build-tools/
   - "Desktop development with C++" 워크로드 선택
2. 컴퓨터 재부팅 후 pip 재시도

### 문제 3: Discord 토큰 오류
**증상:** `❌ 오류: Improper token has been passed`

**해결방법:**
1. `.env` 파일 확인:
   ```powershell
   Get-Content .env
   ```
2. Discord Developer Portal에서 토큰 재생성:
   - https://discord.com/developers/applications
   - 봇 선택 → Bot → Reset Token
3. 새 토큰을 `.env` 파일에 저장

### 문제 4: 봇이 백그라운드에서 시작되지 않음
**해결방법:**
1. 작업 스케줄러 상태 확인:
   ```powershell
   Get-ScheduledTask -TaskName "HoyoBot_AutoStart"
   ```
2. 에러 로그 확인:
   ```powershell
   Get-Content bot_error.log -Tail 50
   ```
3. 수동 실행으로 테스트:
   ```powershell
   python main.py
   ```

### 문제 5: 여러 Python 프로세스가 실행됨
**해결방법:**
```powershell
# 모든 Python 프로세스 종료
taskkill /F /IM python.exe

# 봇 재시작
.\scripts\control_bot.ps1
```
