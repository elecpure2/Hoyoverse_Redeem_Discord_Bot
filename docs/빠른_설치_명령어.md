# 🚀 빠른 설치 명령어 모음

새 컴퓨터에서 호요봇을 설치하기 위한 **복사-붙여넣기용** 명령어 모음입니다.

---

## 📋 1단계: Python 설치 확인

```powershell
# Python 버전 확인
python --version

# pip 버전 확인
pip --version
```

**Python이 설치되지 않았다면:**
- https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe
- 설치 시 "Add Python to PATH" 반드시 체크!

---

## 📦 2단계: 프로젝트 폴더로 이동 및 패키지 설치

```powershell
# 프로젝트 폴더로 이동
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"

# pip 업그레이드
python -m pip install --upgrade pip

# 필수 패키지 설치
pip install -r requirements.txt

# 선택적 패키지 설치 (Hakushin)
pip install git+https://github.com/thesadru/hakushin.git
```

---

## 🔑 3단계: 환경 변수 설정

```powershell
# .env 파일이 없으면 생성
if (!(Test-Path .env)) {
    Copy-Item .env.example .env
}

# .env 파일 열기
notepad .env
```

**메모장에서 다음 형식으로 입력:**
```
DISCORD_TOKEN="실제_디스코드_봇_토큰"
GEMINI_API_KEY="실제_Gemini_API_키"
```

---

## ✅ 4단계: 봇 테스트 실행

```powershell
# 봇 수동 실행 (테스트)
python main.py
```

**정상 작동 확인 후 Ctrl+C로 종료**

---

## 🤖 5단계: 백그라운드 서비스 설정

### PowerShell을 **관리자 권한**으로 실행 후:

```powershell
# 프로젝트 폴더로 이동
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"

# 서비스 설치
.\scripts\install_service.ps1

# 봇 제어 패널 실행
.\scripts\control_bot.ps1
```

제어 패널에서 **"1. 봇 시작"** 선택

---

## 🎮 봇 제어 명령어

```powershell
# 봇 제어 패널 (추천)
.\scripts\control_bot.ps1

# 수동 실행 (로그 확인용)
python main.py

# 로그 실시간 보기
Get-Content bot.log -Wait -Tail 50

# 봇 프로세스 강제 종료
taskkill /F /IM python.exe
```

---

## 🔧 문제 해결 명령어

```powershell
# 설치된 패키지 확인
pip list

# 작업 스케줄러 상태 확인
Get-ScheduledTask -TaskName "HoyoBot_AutoStart"

# 에러 로그 확인
Get-Content bot_error.log -Tail 50

# Python PATH 확인
$env:PATH -split ';' | Select-String python

# 서비스 제거 (필요시)
.\scripts\uninstall_service.ps1
```

## 📝 전체 설치 과정 (한 번에 복사)

**일반 PowerShell에서 실행:**
```powershell
# 프로젝트 폴더로 이동
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"

# pip 업그레이드
python -m pip install --upgrade pip

# 패키지 설치
pip install -r requirements.txt
pip install git+https://github.com/thesadru/hakushin.git

# .env 파일 생성 (없는 경우)
if (!(Test-Path .env)) {
    Copy-Item .env.example .env
    Write-Host "⚠️ .env 파일을 열어서 토큰과 API 키를 입력하세요!" -ForegroundColor Yellow
    notepad .env
}

# 봇 테스트 실행
Write-Host "`n✅ 설치 완료! 봇을 테스트 실행합니다..." -ForegroundColor Green
Write-Host "정상 작동 확인 후 Ctrl+C로 종료하세요.`n" -ForegroundColor Cyan
python main.py
```

**관리자 PowerShell에서 실행 (백그라운드 서비스 등록):**
```powershell
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"
.\scripts\install_service.ps1
.\scripts\control_bot.ps1
```

## ⚡ 원클릭 설치 스크립트

아래 내용을 `quick_setup.ps1`로 저장하고 실행:

```powershell
# 호요봇 빠른 설치 스크립트
Write-Host "🚀 호요봇 설치를 시작합니다...`n" -ForegroundColor Green

# Python 확인
Write-Host "1️⃣ Python 확인 중..." -ForegroundColor Cyan
try {
    $pythonVersion = python --version 2>&1
    Write-Host "   ✅ $pythonVersion" -ForegroundColor Green
} catch {
    Write-Host "   ❌ Python이 설치되지 않았습니다!" -ForegroundColor Red
    Write-Host "   다음 링크에서 Python 3.11을 다운로드하세요:" -ForegroundColor Yellow
    Write-Host "   https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe`n" -ForegroundColor Yellow
    exit 1
}

# 프로젝트 폴더 확인
Write-Host "`n2️⃣ 프로젝트 폴더 확인 중..." -ForegroundColor Cyan
$projectPath = "C:\Users\elecp\Desktop\Server\Hoyo Bot"
if (Test-Path $projectPath) {
    Set-Location $projectPath
    Write-Host "   ✅ 프로젝트 폴더 발견: $projectPath" -ForegroundColor Green
} else {
    Write-Host "   ❌ 프로젝트 폴더를 찾을 수 없습니다!" -ForegroundColor Red
    exit 1
}

# pip 업그레이드
Write-Host "`n3️⃣ pip 업그레이드 중..." -ForegroundColor Cyan
python -m pip install --upgrade pip --quiet
Write-Host "   ✅ pip 업그레이드 완료" -ForegroundColor Green

# 패키지 설치
Write-Host "`n4️⃣ 필수 패키지 설치 중..." -ForegroundColor Cyan
pip install -r requirements.txt --quiet
Write-Host "   ✅ 필수 패키지 설치 완료" -ForegroundColor Green

Write-Host "`n5️⃣ 선택적 패키지 설치 중 (Hakushin)..." -ForegroundColor Cyan
try {
    pip install git+https://github.com/thesadru/hakushin.git --quiet
    Write-Host "   ✅ Hakushin 설치 완료" -ForegroundColor Green
} catch {
    Write-Host "   ⚠️ Hakushin 설치 실패 (선택사항이므로 무시)" -ForegroundColor Yellow
}

# .env 파일 확인
Write-Host "`n6️⃣ 환경 변수 파일 확인 중..." -ForegroundColor Cyan
if (!(Test-Path .env)) {
    Copy-Item .env.example .env
    Write-Host "   ⚠️ .env 파일이 생성되었습니다!" -ForegroundColor Yellow
    Write-Host "   Discord 토큰과 Gemini API 키를 입력해야 합니다." -ForegroundColor Yellow
    
    $openEnv = Read-Host "`n   지금 .env 파일을 열어서 설정하시겠습니까? (y/n)"
    if ($openEnv -eq 'y') {
        notepad .env
        Write-Host "`n   .env 파일을 저장하고 닫은 후 Enter를 누르세요..." -ForegroundColor Cyan
        Read-Host
    }
} else {
    Write-Host "   ✅ .env 파일이 이미 존재합니다" -ForegroundColor Green
}

# 설치 완료
Write-Host "`n✅ 설치가 완료되었습니다!`n" -ForegroundColor Green
Write-Host "다음 단계:" -ForegroundColor Cyan
Write-Host "  1. 봇 테스트 실행: python main.py" -ForegroundColor White
Write-Host "  2. 백그라운드 서비스 등록 (관리자 PowerShell):" -ForegroundColor White
Write-Host "     .\scripts\install_service.ps1" -ForegroundColor White
Write-Host "  3. 봇 제어: .\scripts\control_bot.ps1`n" -ForegroundColor White

$testRun = Read-Host "지금 봇을 테스트 실행하시겠습니까? (y/n)"
if ($testRun -eq 'y') {
    Write-Host "`n🤖 봇을 시작합니다... (종료하려면 Ctrl+C)`n" -ForegroundColor Green
    python main.py
}
```
