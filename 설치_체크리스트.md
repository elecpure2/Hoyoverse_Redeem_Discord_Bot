# 🚀 호요봇 설치 체크리스트

새 컴퓨터에서 봇을 실행하기 위한 단계별 가이드입니다.

---

## ✅ 1단계: Python 3.11 설치

### 다운로드 링크:
**Python 3.11.9 (64-bit):**
https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe

### 설치 방법:
1. 위 링크를 브라우저에서 열어 다운로드
2. 다운로드한 파일 실행
3. ⚠️ **중요:** "Add Python 3.11 to PATH" 체크박스 **반드시 선택**
4. "Install Now" 클릭
5. 설치 완료 후 PowerShell **완전히 종료 후 재시작**

### 설치 확인:
```powershell
python --version
```
**예상 출력:** `Python 3.11.9`

---

## ✅ 2단계: 필수 패키지 설치

PowerShell에서 프로젝트 폴더로 이동 후 실행:

```powershell
cd "C:\Users\elecp\Desktop\Server\Hoyo Bot"

# pip 업그레이드
python -m pip install --upgrade pip

# 필수 패키지 설치
pip install -r requirements.txt

# 선택적 패키지 (Hakushin)
pip install git+https://github.com/thesadru/hakushin.git
```

## ✅ 3단계: Discord 봇 토큰 생성

### Discord Developer Portal:
https://discord.com/developers/applications

### 토큰 생성 방법:
1. 위 링크 접속 후 로그인
2. "New Application" 클릭
3. 봇 이름 입력 (예: Hoyo Bot)
4. 왼쪽 메뉴에서 "Bot" 클릭
5. "Reset Token" 클릭 → 토큰 복사 (⚠️ 한 번만 보여짐!)
6. 토큰을 안전한 곳에 저장

### 봇 권한 설정:
1. "Privileged Gateway Intents" 섹션에서:
   - ✅ MESSAGE CONTENT INTENT 활성화
2. 왼쪽 메뉴에서 "OAuth2" → "URL Generator"
3. Scopes: `bot`, `applications.commands` 선택
4. Bot Permissions: `Administrator` 선택 (또는 필요한 권한만)
5. 생성된 URL로 봇을 서버에 초대

https://aistudio.google.com/app/apikey

1. 위 링크 접속 후 Google 계정으로 로그인

## ✅ 5단계: .env 파일 설정

### .env 파일 생성 및 편집:
```powershell
# 프로젝트 폴더에서
notepad .env
```

### .env 파일 내용:
```env
DISCORD_TOKEN="여기에_3단계에서_복사한_디스코드_토큰_붙여넣기"
GEMINI_API_KEY="여기에_4단계에서_복사한_Gemini_API_키_붙여넣기"
```

**예시:**
```env
DISCORD_TOKEN="MTI3...(여기에 실제 봇 토큰, 약 70자)...rStUv"
GEMINI_API_KEY="AIzaSy...(여기에 실제 Gemini 키)...PpQq"
```

⚠️ **주의:** 따옴표 안에 실제 토큰/키를 입력하세요!

## 🆘 문제 해결

### Python 명령어가 안 됨
- PowerShell 완전히 종료 후 재시작
- 컴퓨터 재부팅
- Python 재설치 시 "Add to PATH" 확인

### 패키지 설치 오류
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 봇이 시작되지 않음
- `.env` 파일의 토큰/키 확인
- Discord 봇이 서버에 초대되었는지 확인
- `bot.log` 파일에서 오류 확인
