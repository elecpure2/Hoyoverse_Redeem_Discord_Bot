# 🤖 Hoyoverse Redeem & Info Discord Bot

호요버스 게임(원신 · 붕괴: 스타레일 · 젠레스 존 제로 · 명일방주: 엔드필드)용 **리딤코드 알림 + 게임 정보 조회** 디스코드 봇입니다.

> **데이터 출처:** 게임 정보(캐릭터/무기/성유물/출시예정/업데이트)는 [`nanoka.cc`](https://nanoka.cc) (구 hakush.in, 폐쇄 후 부활)에서 가져옵니다.

---

## ✨ 주요 기능

- **🎁 리딤코드 자동 알림** — 원신 / 스타레일 / 젠레스 / 엔드필드 새 코드 감지 시 채널에 알림
- **📺 유튜브 · 커뮤니티 알림** — 등록 채널의 새 영상/커뮤니티 글 알림
- **🔍 게임 정보 조회** — 캐릭터 · 무기/광추 · 성유물/유물/디스크 통합 검색 (3게임)
- **🆕 출시 예정 + 업데이트 알람** — 신규 캐릭터/무기/성유물, nanoka 데이터 업데이트 알림
- **💫 기원 시뮬레이터** — 천장/픽뚫 포함 1~100회, 결과를 동그라미 그리드로 시각화
- **🔮 오늘의 운세** — 캐릭터별 인게임 스토리 기반 운세 (하루 1회)
- **🎮 Enka 빌드 조회** — UID로 전시 캐릭터 빌드 확인
- **💬 AI 캐릭터 대화** — Gemini 기반 캐릭터 페르소나 대화

---

## 📋 명령어

`/` 슬래시 또는 `!` 접두사 모두 사용 가능합니다.

| 명령어 | 설명 |
|---|---|
| `!운세` | 오늘의 운세 (하루 1회) |
| `!기원` · `!기원 10` · `!기원 100` | 기원 시뮬 (1~100회) · `!기원리셋` |
| `!캐릭터 이름` · `!무기 이름` · `!성유물 이름` | 게임 정보 조회 |
| `!신캐` · `!신무기` · `!신성유물` | 출시 예정 콘텐츠 |
| `!uid 숫자` · `!빌드 캐릭터` | Enka 빌드 |
| `!이벤트 게임명` | 진행 중 이벤트 |
| `!푸리나` · `!콜롬비나` · `!캐서린` | AI 캐릭터 대화 |
| `!도움` | 명령어 목록 |
| *(관리자)* `!알림설정` · `!알림해제` · `!알림현황` | 채널 알림 설정 |
| *(관리자)* `!업데이트상태` · `!커뮤확인` · `!영상확인` | 진단 |

## ⚙️ 설치 & 실행

### 요구사항
- **Python 3.11** (설치 시 "Add Python to PATH" 체크)
- 패키지 설치:
  ```bash
  pip install -r requirements.txt
  ```
  > `hakushin` 패키지(게임 Enum용)가 필요하면: `pip install hakushin`

### 환경 변수 (`.env`)
프로젝트 루트에 `.env` 파일을 만들고 입력 (`.env.example` 참고):
```env
DISCORD_TOKEN="디스코드_봇_토큰"
GEMINI_API_KEY="Gemini_API_키"      # AI 대화 기능용 (선택)
YOUTUBE_API_KEY="유튜브_API_키"      # 선택 (RSS 모드면 불필요)
```
> ⚠️ `.env` 와 `data/*.json`(서버/사용자 데이터)은 `.gitignore`에 포함되어 깃에 올라가지 않습니다.

### 실행
```powershell
# 로그 보며 실행
.\run.ps1
# 또는
python main.py
```
백그라운드/자동 시작은 `scripts/` 의 스크립트를 참고하세요.
